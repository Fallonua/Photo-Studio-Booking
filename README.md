# Photo Studio Booking

Настольное приложение для учёта и бронирования залов фотостудии.
Администратор создаёт брони за клиентов, управляет залами и оборудованием,
просматривает расписание занятости и формирует отчёты по загрузке.

Реализовано на **Java 17 + JavaFX**, данные хранятся в локальной встроенной БД **SQLite**,
сборка — через **Apache Maven**.

---

## Возможности

- вход администратора по паролю;
- просмотр каталога залов (название, вместимость, цена/час, описание интерьера);
- создание брони для клиента (имя, телефон, дата, время, дополнительное оборудование);
- автоматический расчёт стоимости (зал + оборудование × длительность);
- просмотр расписания зала на дату (занятые и свободные слоты 09:00–21:00);
- просмотр всех бронирований, редактирование активных и отмена;
- управление залами и оборудованием (добавление, изменение, удаление);
- отчёт по загрузке залов за период (число броней, часы, выручка);
- выход из системы с возвратом на экран входа.

---

## Требования

- **JDK 17 или выше**;
- **Apache Maven 3.8+**;
- постоянное подключение к интернету **не требуется** — приложение работает автономно.

Проверить, что всё установлено:

```bash
java -version
mvn -v
```

## Установка и запуск

### 1. Установить JDK 17+

- **Windows:** скачайте установщик с [adoptium.net](https://adoptium.net/) и установите.
- **Linux:** `sudo apt install openjdk-17-jdk` (или аналог для вашего дистрибутива).

### 2. Установить Apache Maven

- Скачайте с [maven.apache.org](https://maven.apache.org/download.cgi),
распакуйте и добавьте папку `bin` в переменную окружения `PATH`.

### 3. Собрать проект

В корневой папке проекта (где лежит `pom.xml`) выполните:

```bash
mvn clean package
```

Команда скомпилирует код и запустит unit-тесты. При успешной сборке
артефакты появятся в папке `target/`.

### 4. Запустить приложение

```bash
mvn clean javafx:run
```

При первом запуске в рабочей директории автоматически создаётся файл базы данных
`studio.db` с демонстрационными данными — отдельно устанавливать СУБД не нужно.

## Первые шаги

1. На экране **«Photo Studio — Вход»** введите пароль `**admin`** и нажмите **«Войти»**.
2. Перейдите на вкладку **«Создать бронь»**, выберите зал и нажмите **«Выбрать и забронировать»**.
3. Заполните имя и телефон клиента, выберите дату и время из доступных слотов,
  при необходимости отметьте оборудование и подтвердите бронь.
4. На вкладке **«Бронирования»** просмотрите созданную запись;
  на вкладке **«Отчёт по загрузке»** сформируйте статистику за нужный период.

Подробнее — в [руководстве пользователя](/docs/user-guide.md).

## Бизнес-ограничения

- дата брони — не в прошлом и не больше, чем на 30 дней вперёд;
- начало брони на сегодня — не ранее, чем через 30 минут от текущего момента;
- длительность — от 1 до 8 часов;
- рабочее время студии: **09:00–21:00**;
- создание брони выполняется в транзакции (бронь + связи с оборудованием).

## Тестирование

Запуск unit-тестов сервисного слоя:

```bash
mvn test
```

Тесты покрывают расчёт стоимости, валидацию дат и времени, проверку доступных слотов
и аутентификацию администратора.
